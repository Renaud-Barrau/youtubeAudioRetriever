Downloads all the videos in a playlist and extracts their audio. 

This method takes four input parameters\+:
\begin{DoxyItemize}
\item video\+Title\+Array\+: A QVector of QString containing the titles of the videos in the playlist.
\item video\+Id\+Array\+: A QVector of QString containing the IDs of the videos in the playlist.
\item video\+Thumbnails\+Url\+Array\+: A QVector of QString containing the URLs of the thumbnails of the videos in the playlist.
\item playlist\+Name\+: A QString containing the name of the playlist.

It first checks if the playlist folder already exists, and if not, it creates a new folder with the given playlist name. Then, it gets a list of all the files in the playlist folder. It calculates the progress bar increment value based on the number of videos in the playlist. It then loops through the video\+Title\+Array and checks if the corresponding video file already exists in the playlist folder. If the file exists, it increments the progress bar value. If the file does not exist, it calls the \doxylink{classyoutube_downloader_a9fba11b5e23ebc4cea08150b1cc94db2}{format\+Command()} method to create the command string for extracting and downloading the audio of the video. It then adds the command string to the download\+Queue queue. Finally, if the download\+Queue is not empty, it calls the \doxylink{classyoutube_downloader_ae2ca25de6505a546ff66698e45eb6f1b}{exec\+Command()} method to start the first command in the queue.
\end{DoxyItemize}

Executes the command to extract and download audio from a You\+Tube video using the yt-\/dlp.\+exe executable in the background. 

This method takes one input parameter\+: command, which is the command string constructed by the \doxylink{classyoutube_downloader_a9fba11b5e23ebc4cea08150b1cc94db2}{format\+Command()} method. It creates a new QProcess object and starts the command using the start\+Command() method. It then connects the finished() signal of the QProcess object to a lambda function that deletes the QProcess object and updates the progress bar. The progress bar is updated by incrementing the progress\+Bar\+Value member variable. The \doxylink{classyoutube_downloader_a118d28eadd74aaa9131519ef5c7d9f3d}{start\+Next\+Download()} method is then called to start the next download in the queue, if any.

Formats the command to extract and download audio from a You\+Tube video using the yt-\/dlp.\+exe executable. 

This method takes three input parameters\+: video\+Title, video\+Id, and playlist\+Name. It constructs a command string using these input parameters and stores it in the command member variable. The command string is constructed as follows\+:


\begin{DoxyItemize}
\item "{}yt-\/dlp.\+exe"{} \+: the name of the executable to run
\item "{}-\/x"{} \+: extract audio from the video
\item "{}-\/-\/audio-\/format mp3"{} \+: set the audio format to mp3
\item "{}-\/-\/audio-\/quality 192"{} \+: set the audio quality to 192 kbps
\item "{}-\/-\/write-\/thumbnail"{} \+: write the video thumbnail to the output file
\item "{}-\/-\/ffmpeg-\/location"{} \+: the path to the ffmpeg executable, which is used by yt-\/dlp.\+exe to extract and encode audio
\item "{}-\/\+P"{} \+: the path to the output directory, which is specified as a combination of the music\+Folder and playlist\+Name member variables
\item "{}-\/o"{} \+: the output file name, which is specified as the video\+Title input parameter
\item "{}https\+://www.\+youtube.\+com/watch?v="{} \+: the URL of the You\+Tube video, which is specified as a combination of the "{}https\+://www.\+youtube.\+com/watch?v="{} string and the video\+Id input parameter
\end{DoxyItemize}

The command string is then printed to the console using cout.
